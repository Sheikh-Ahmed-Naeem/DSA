# Data Structures and Algorithms 

This repository contains implementations of various data structures and algorithms in C++.

## Contents

- **Data Structures**:
    Below is a list of data structures implemented in this repository:

    - LinkedList
    - Stack
    - Queue
    - Binary Search Tree
    - Heap
    - Hash Table
    - Graph
- **Algorithms**: 
 Below is a list of algorithms implemented in this repository:

- Sorting Algorithms:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
- Searching Algorithms:
  - Linear Search
  - Binary Search
- Graph Algorithms:
  - Breadth-First Search (BFS)
  - Depth-First Search (DFS)
- Dynamic Programming:
  - Fibonacci Series
  - Knapsack Problem
  - Longest Common Subsequence
